# 100days_challenge

## ルール
- 100日間連続でコーディングをする。
  - コーディング課題、個人開発など適宜GPTと相談して内容を決める
  - Python、React、機械学習、深層学習をメインに取り組む 
- 1日1コミット
  - Issue → Branch → PR → Merge の 4 ステップを毎日繰り返す
- 毎週日曜日には1週間の振り返りを行う
  - Issue：week‑N‑retrospective
    1. 達成度 (0‑5)
    2. 学んだこと／詰まったこと
    3. 翌週の取り組み目標
